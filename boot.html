The Monolith

i486 Linux live system — running entirely in your browser

This boots a full Linux system using an x86 CPU emulator (v86/WebAssembly) directly in your browser tab — no plugins, no installs, no server-side execution. The ISO is fetched on demand via HTTP range requests, so only sectors the boot sequence actually reads are downloaded.

~135 MB ISO • 256 MB emulated RAM • i486 kernel
Expect boot to take 1–3 minutes depending on your connection.
